### Sobre esse Projeto

Esse projeto foi o resultado de um Hackathon incentivado no Bootcamp da Generation Brasil.
  A ideia foi criar uma plataforma que armazene os Projetos Integradores da Generation Brasil.

### Funcionalidades

Plataforma capaz de ver projeto integradores, adicionar, editar e excluir. (CRUD)
